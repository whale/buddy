Known deltas & recommended follow-ups

Honest list of what’s not pixel-identical yet, and small decisions for you.

  • Done words. Mac rotates a word per task (“Shipped!”, “Locked In!”); iPhone shows a fixed “Donezo.”. Easy to port the word list next.
  • Row interaction. With no hover on iPhone, a tap completes a task and long-press opens edit/sleep/remove. Clean at rest like the Mac — but confirm this feels right vs. a swipe.
  • “Now/focused” state. The Mac removed it (tap just toggles done); the iPhone store still models it but the UI no longer highlights it. Aligning the store is a small functional PR (3 tests reference it).
  • Weather. A static moon glyph stands in; the live Open-Meteo fetch isn’t ported to iOS yet.
  • Morning done rows. Mac shows completed tasks in the planner too; iPhone currently lists active only.
  • Row size/height. Mac rows stretch to fill the tall drawer; iPhone uses comfortable fixed heights — a judgement call, tune if you want them taller.
